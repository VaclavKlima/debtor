# Debtor version 2.0

##### Instalation:

Run commands:

- `composer install`
- `yarn install`
- `yarn prod`
- `php artisan migrate`

------------

##### Create new admin user

Run command `php artisan debtor:create-new-user {email}`

Email with password reset link will be sent to the email.
